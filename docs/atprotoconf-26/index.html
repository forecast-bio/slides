# Python
@atdata.packable
class DrugResponse:
    compound_id: str
    dose_um: float
    viability: float
    response_curve: NDArray
// Rust
#[derive(Packable)]
struct DrugResponse {
    compound_id: String,
    dose_um: f64,
    viability: f64,
    response_curve: NdArray,
}

The atdata API

HuggingFace-style, but federated

Load any dataset from any PDS with a single call.

Resolve by handle + name
Stream shards on demand
Schema-aware deserialization

Works with labels:
@forecast.bio/mnist resolves to the latest entry for that dataset on Forecast's PDS.

import atdata

# Load by handle + dataset name
ds = atdata.load_dataset(
    "@forecast.bio/mnist"
)

for sample in ds.shuffled():
    print(sample.label)
    print(sample.image.shape)
use atdata::Dataset;

let ds = Dataset::load(
    "@forecast.bio/mnist"
).await?;

for sample in ds.shuffled() {
    println!("{}", sample.label);
}

Under the hood

How data moves through the network

WebDataset

Sharded tar archives — streaming, S3-optimal, append-only. One shard = one worker.

@packable

Typed sample classes in Python & Rust. Drives schema generation + serialization automatically.

Array formats

ndarray · sparse · Arrow · safetensors · Parquet — with JSON Schema shims for each.
